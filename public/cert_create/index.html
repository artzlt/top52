Fasttemplate for PHP 4.x, 5.x

This version of Fasttemplate is adopted for PHP 4.x, including dynamic template handling.
Tested with PHP 4.x.x (WinNT)

Samples:

  • example_2.php
  • example_3.php
  • flex_table.php
  • phpcode_with_debug.php
    Example to see how to show a debug windows, to see different Fast Template errors.
  • Cache with Fast Template / Delete expired cache files (new in 1.3.0)
    Write/Delete content into a cache file to make faster load of the page second time, until cache is valid.
  • FastWrite
    It is like FastPrint, just write content into a file. Very usefull for making offline content,
    write different outputs into files etc.
  • Language DEMO for multiple_assign (same method can be used if you use define instead of $)
    It is easy now to simple use a in template a {TAG) and to be replaced direcctly from
    language file for example, without an assign. See the demo file and source.
  • Added support for access object fields using get() or getId() methods from templates
    (For ex. {Object.field} = Object->get('field') && {Object.id} = Object->getId() )
    It is something like using JavaBeans in JSP
  • Added FastTemplate::parse_and_return($template_name) method
    Parse template and return it as string Can be used instead FastTemplate::parse() && FastTemplate::fetch
    Added templates comments. Smth like {* This is a comment *}
    New FastTemplate::define() - now supports array and name and value as parameter
  • Added <!--#include file="filename.html" --> template statement (new in 1.3.0)
    It is usefull if you have several different templates for different parts of page and you dont need to write
    any php code to gather all "blocks" of the page
